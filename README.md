# Harun Ahmed — Graphics Designer Portfolio

A personal portfolio website for Harun Ahmed, a professional Graphics Designer specializing in custom clothing designs, menus, advertisements, and branding.

🌐 **Live Site:** [harun-ahmed-portfolio.vercel.app](https://harun-ahmed-portfolio.vercel.app)

---

## Tech Stack

| Technology                 | Purpose                     |
| -------------------------- | --------------------------- |
| React 18                   | UI framework                |
| Vite                       | Build tool & dev server     |
| TypeScript                 | Type safety                 |
| Tailwind CSS               | Styling                     |
| shadcn/ui                  | UI component library        |
| EmailJS                    | Contact form email delivery |
| Lucide React & React Icons | Icons                       |
| Sonner                     | Toast notifications         |
| Vercel                     | Hosting & deployment        |

---

## Features

- Responsive design optimized for all screen sizes
- Work/project showcase gallery
- Contact form with direct email delivery via EmailJS
- Social media links (TikTok, Telegram, Instagram)
- Dark mode support
- Smooth animations and hover interactions
